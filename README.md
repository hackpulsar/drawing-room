## Build instructions
```
git clone --recurse-submodules https://github.com/hackpulsar/drawing-room
cd drawing-room
cmake -S. -B/bin
cd bin
make
```
This will generate ```/client``` and ```/server``` directories. You will find binaries for client and server there.
